# bclk

Binary Clock for Desktop.
If you don't know how to read it, please check [wikipedia](https://en.wikipedia.org/wiki/Binary_clock).

![Screenshot](./images/bclk.png)

## Build and Run

```bash
git clone git@github.com:joaoofreitas/bclk.git
cd bclk
zig build run
```
